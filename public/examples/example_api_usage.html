WOLFIE Headers v2.0.8 - API Examples

MAAT's Balance Note: All API endpoints are in harmony. Directory-level logs (markdown files) and row-level logs (database _logs tables) complement each other perfectly. v2.0.8 adds shared hosting compatibility using SHOW TABLES and DESCRIBE instead of information_schema.

1. Discover _logs Tables

GET /api/wolfie/logs/tables

Discover all _logs tables in the database.

fetch('/api/wolfie/logs/tables')
  .then(response => response.json())
  .then(data => {
    console.log('Discovered tables:', data.data);
    console.log('Total tables:', data.metadata.total_tables);
  });

2. Get Change Logs for Row

GET /api/wolfie/logs/{table_name}/{row_id}

Get all change log entries for a specific database row.

// Get change logs for content row 123
fetch('/api/wolfie/logs/content/123?limit=20&agent_id=8')
  .then(response => response.json())
  .then(data => {
    console.log('Change logs:', data.data.change_logs);
    console.log('Summary:', data.data.summary);
  });

3. Write Change Log Entry

POST /api/wolfie/logs/{table_name}/{row_id}

Write a new change log entry to track row-level changes.

fetch('/api/wolfie/logs/content/123', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    agent_id: 8,
    agent_name: 'WOLFIE',
    channel_id: 7,
    change_data: {
      change_type: 'update',
      changed_fields: ['title', 'status'],
      old_values: {
        title: 'Old Title',
        status: 'draft'
      },
      new_values: {
        title: 'New Title',
        status: 'published'
      }
    },
    metadata: {
      change_reason: 'User requested title update',
      change_summary: 'Updated title and published content'
    }
  })
})
  .then(response => response.json())
  .then(data => {
    console.log('Change log created:', data.data);
  });

4. List Change Logs for Table

GET /api/wolfie/logs/{table_name}

List all change log entries for a table (across all rows).

// List all change logs for content table
fetch('/api/wolfie/logs/content?limit=50&channel_id=7')
  .then(response => response.json())
  .then(data => {
    console.log('Change logs:', data.data);
    console.log('Total results:', data.metadata.total_results);
  });

Complete Example

Complete workflow: Discover tables, write change log, read change logs.

// 1. Discover _logs tables
const tables = await fetch('/api/wolfie/logs/tables')
  .then(r => r.json())
  .then(d => d.data);

console.log('Available tables:', tables);

// 2. Write a change log entry
const logEntry = await fetch('/api/wolfie/logs/content/123', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    agent_id: 8,
    agent_name: 'WOLFIE',
    channel_id: 7,
    change_data: {
      change_type: 'update',
      changed_fields: ['title'],
      old_values: { title: 'Old' },
      new_values: { title: 'New' }
    }
  })
}).then(r => r.json());

console.log('Created log entry:', logEntry.data);

// 3. Read change logs
const logs = await fetch('/api/wolfie/logs/content/123')
  .then(r => r.json())
  .then(d => d.data);

console.log('Change logs:', logs.change_logs);
console.log('Summary:', logs.summary);
